# Discoverability Checklist

Use this checklist to make the public GitHub project easier to find.

## GitHub About

Suggested short description:

```text
Simple CLI starter that creates PRD, memory, tasks, notes, and agent instructions for AI-assisted coding projects.
```

Suggested website:

```text
https://github.com/shebekocom/unit-agent-starter#readme
```

## GitHub Topics

GitHub topics help people discover repositories by subject. GitHub recommends lowercase topics with hyphens, 50 characters or less, and no more than 20 topics.

Recommended topics:

```text
ai-agent
agent-starter
agents-md
claude-code
codex
gemini
vibe-coding
project-template
starter-template
developer-tools
cli
npm-package
prd
ai-tools
agentic-workflow
```

## README Keywords

The README should naturally include phrases people may search:

- AI agent starter
- vibe coding starter
- Claude Code starter
- Codex AGENTS.md starter
- Gemini project instructions
- PRD memory tasks notes
- simple project memory for AI agents

## Places To Share

- GitHub topics pages
- npm package page
- SkillsMP / skill communities when relevant
- Awesome AI agents lists
- Reddit / Hacker News / Indie Hackers after the README is polished
- personal site: prom.help

## Launch Notes

Before sharing widely:

- publish npm package;
- add a demo GIF or terminal screenshot;
- add 2-3 example generated projects;
- add a clear comparison: "not an agent framework, just a tiny project brain";
- create issues labeled `good first issue`.
